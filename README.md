This is image downloader for remote files

## Install

Via Composer

``` bash
$ composer require :artemukolov/:imagedownloader
```

## Usage

``` php
//try to get local path of downloaded
$result = \ArtemUkolov\ImageDownloader\Downloader::get("http://ukolov.info/img/me.jpg");
```
